# AI-Based Study Planner

## Project Description

AI-Based Study Planner is a simple Java Swing application that helps students create a study plan for their exams.

The user enters information such as student name, roll number, subject, difficulty level, priority, exam date, study hours, and progress notes. The system validates the input and generates a study recommendation based on the selected difficulty level.

This project demonstrates:

* Event Handling
* Exception Handling
* Code Refactoring
* Unit Testing using JUnit 4
* Git and GitHub Version Control

## Features

* Student information form
* Difficulty level selection
* Priority selection using radio buttons
* Input validation
* Study plan generation
* Reset form option
* Exit application option
* JUnit 4 test cases

## Technologies Used

* Java
* Java Swing
* JUnit 4
* NetBeans IDE

## Project Files

* `JFrame.java` - Main graphical user interface
* `Classs.java` - Validation and recommendation methods
* `ClasssTest.java` - JUnit 4 test cases

## Setup Instructions

### Requirements

* Java JDK 8 or later
* NetBeans IDE
* JUnit 4 Library

### Steps to Run

1. Download or clone the project.
2. Open the project in NetBeans.
3. Make sure JUnit 4 library is added.
4. Run `JFrame.java`.
5. Enter the required information.
6. Click **Generate Plan** to create a study plan.

### Running Unit Tests

1. Open `ClasssTest.java`.
2. Right-click the file.
3. Select **Test File**.
4. All test cases should pass successfully.
